# CodeAlpha DevOps Internship - Task 4: Web Server using Docker

## Overview
This repository contains the solution for Task 4 of the CodeAlpha DevOps Internship. The objective of this task is to set up a containerized Nginx web server using Docker.

## Prerequisites
- Docker installed on your host machine (e.g., Ubuntu).
- Basic understanding of Linux command line.

## Project Structure
- `index.html`: A simple custom HTML page served by the web server.
- `Dockerfile`: The configuration file containing instructions to build the Docker image.
